# GA1-220501096-03-AA1-EV07 — Reutilización de componentes en React

## Descripción

Esta actividad consiste en construir una hoja de vida dinámica utilizando React y Vite. Se aplicaron props y desestructuración para reutilizar componentes de forma eficiente y estilizada.

## Archivos incluidos

- `hoja-de-vida-react-props.docx`: documento explicativo con estructura del proyecto, componentes utilizados y justificación técnica.
- Código fuente en la carpeta `react-vite-lab` (si aplica).

## Componentes utilizados

- `App.jsx`: componente raíz que organiza la estructura general.
- `CabeceraCV.jsx`: componente reutilizable que recibe props.
- `cv.css`: hoja de estilos con diseño moderno.

## Objetivo

Demostrar el uso de props y desestructuración en componentes React, aplicando buenas prácticas de modularidad, reutilización y diseño limpio.

## Tecnologías

- React
- Vite
- CSS
